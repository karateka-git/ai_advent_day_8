# Overview

Перед началом работы в этом проекте стоит просматривать локальный `MemoryBank`, потому что здесь могут быть команды и соглашения, отличающиеся от общих правил.

Локальные правила для этого проекта:
- не править русские строки без отдельной просьбы пользователя;
- в проекте использовать только русские фразы, не заменять их английскими без отдельной просьбы пользователя.
